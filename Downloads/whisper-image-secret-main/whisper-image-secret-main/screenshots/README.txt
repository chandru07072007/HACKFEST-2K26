Screenshots Folder
==================

Please add the following screenshot images to this folder:

1. encode-message.png - Screenshot of the encode message interface
2. decode-message.png - Screenshot of the decode message interface  
3. extracted-message.png - Screenshot showing the extracted message result

These images will be displayed in the main README.md file.

You can either:
- Take new screenshots of the running application
- Save the existing screenshots from your browser/application
- Copy the images you already have and rename them accordingly
